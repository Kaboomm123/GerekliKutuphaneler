# GerekliKutuphaneler
projelerde kullanılan yüklenmesi gereken kütüphaneler

customtkinter: Modern görünümlü grafik arayüz (GUI) tasarımı için.

requests: Gist ve API'lerden veri çekmek için (senkron).

aiohttp: Hızlı bakiye sorgulama işlemleri için (asenkron).

pandas: Veri analizi ve tablo oluşturma için.

xlsxwriter: Excel dosyalarını detaylı formatlandırmak için (engine).

openpyxl: Excel dosyalarına veri yazmak ve hücre birleştirmek için.

Pillow (PIL): Arayüzde logo ve ikon kullanımı için.

base58: Tron adres dönüşümleri için.

tkcalendar: Tarih seçimi (takvim) arayüzü için.


Yüklemek için gerekli komut:
pip install customtkinter requests aiohttp pandas xlsxwriter openpyxl Pillow base58 tkcalendar
